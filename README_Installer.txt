COSMIC RAYS - Professional Granular Audio Processor
==============================================
Developed by Very Good
Official Website: https://beechem.site/cosmic-rays

Cosmic Rays is a professional-grade granular processor inspired by boutique 
hardware granular synthesis. It utilizes advanced kinematic motion models and 
a dedicated global modulation engine to create lush, nostalgic, and evolving 
sonic textures.

CORE FEATURES
-------------
- 256-Voice High-Performance Granular Engine.
- Kinematic Grain Motion: Grains follow physics-modeled ballistic trajectories.
- Global Modulation Engine (MDL) for organic "tape drift" and instability.
- 11 Unique Algorithms including Mosaic, Seq, Glide, Haze, and more.
- 12dB/oct Resonant Filter and Diffusion Network.
- Phase-accurate Phrase Looper.

SYSTEM REQUIREMENTS
-------------------
- macOS: 10.15 or later (Universal Binary: Intel & Apple Silicon).
- Windows: 10 or 11 (64-bit).
- Linux: Ubuntu 22.04 or later (or compatible Debian-based distribution).
- DAW: Any host supporting VST3, AU (macOS only), or LV2.

INSTALLATION
------------
This installer will place the plugin files in the standard system locations:
- macOS: /Library/Audio/Plug-Ins/
- Windows: C:\Program Files\Common Files\VST3\
- Linux: /usr/lib/vst3/ or ~/.vst3/
